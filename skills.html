Skills

Here are the technologies and tools I've worked with so far. I believe in learning by building, and each of these skills represents hands-on effort — not just theory.

HTML5

I use HTML5 as the structural foundation of every website I build. I write clean, semantic code that improves accessibility and SEO, and I structure content with purpose — using tags like section, article, nav, and footer effectively.

CSS 3

Styling isn’t just about colors — it’s about experience. I design layouts using Flexbox and Grid systems, implement media queries for mobile responsiveness, and use modern CSS features like variables, transitions, and hover effects to make the UI both usable and elegant.

Responsive Design

Whether it’s a phone, tablet, or desktop — I ensure the websites I build look great across all screen sizes. I follow mobile-first principles and test responsiveness thoroughly using developer tools and real devices.

Animations & Transitions

To make my interfaces come alive, I use keyframe animations and smooth CSS transitions. Whether it's a button hover, a section fade-in, or a card flip — I aim to enhance user experience with subtle motion.

Python

Currently learning Python for backend development. I’m confident with the basics like variables, functions, loops, conditionals, and working on building logic-based projects (like quiz games, calculators, and more). Next up: File handling and OOP.

SQL

Learning SQL to manage and interact with databases. So far, I’ve learned how to perform basic queries like SELECT, INSERT, and UPDATE, and I’m working on understanding JOINs and subqueries. I plan to use this in backend projects with Flask soon

Version Control (Git & GitHub) I use Git to manage changes in my projects and GitHub to showcase my work publicly. I’ve hosted my portfolio and uploaded multiple projects using repositories, commits, and branches — a habit I’m building early to stay organized.
Visual Studio Code My preferred code editor. I’ve customized it with extensions, themes, and shortcuts that speed up my workflow and make coding more efficient.

I'm currently growing my skills in Flask, JavaScript, and C++ (for DSA) — and I'm excited to soon build full-stack web apps that connect the frontend to a real database backend.
